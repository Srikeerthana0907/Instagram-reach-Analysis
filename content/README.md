# Instagram Reach Prediction using Machine Learning

This project analyzes and predicts Instagram post impressions based on engagement metrics such as likes, comments, saves, and textual features (captions & topics).

## Features
- Data cleaning, preprocessing, and standardization
- Exploratory data analysis with visualizations
- Correlation and feature importance analysis
- Machine learning models: Random Forest & XGBoost
- TF-IDF vectorization for caption text
- Reach prediction for new posts

## Tools & Libraries
Python, Pandas, Scikit-learn, XGBoost, Matplotlib, Seaborn
